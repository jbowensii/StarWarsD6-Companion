## Star Wars D6 content for OpenD6 Space on Foundry Virtual Tabletop

This is a Foundry VTT module that adds Star Wars D6 Roleplaying Game content to an OpenD6 Space System game. Star Wars D6 by West End Games is an older system, out of print, and freely available. This is not a full game system, but relies on the [OpenD6 Space System](https://github.com/jbowensii/Foundryvtt-D6Space) (which is also free and released under OpenD6 OGL) in Foundry Virtual Tabletop to work properly. This is a fan-compiled content module based on the fan-produced *Star Wars: Revised, Expanded and Updated* rulebook.

## Requirements

- **FoundryVTT v14** (minimum v14, verified v14.359)
- **OpenD6 Space (od6s) system v3.0.0+**

## Manifest URL

Install in Foundry VTT via **Add-on Modules → Install Module** using this Manifest URL:
```
https://github.com/jbowensii/StarWarsD6-Companion/releases/latest/download/module.json
```

## Features

### One-click Helper Configurator
Apply Star Wars D6 2nd Edition labels and rules to an OpenD6 Space game with a single click in the module settings. This eliminates much of the setup for a new world.

### Star Wars Themed Character Sheets
Two themed actor sheets are included:
- **Star Wars 2nd Edition Theme** — Tan/maroon styling inspired by the Revised, Expanded and Updated rulebook
- **Star Wars 1st Edition Theme** — Classic white/black styling

To use: Open a character sheet → click the sheet icon (top-left of the window header) → select your preferred Star Wars theme.

### Compendium Packs
- Skills & Advanced Skills
- Force Powers and Force Skills
- Character Templates from Star Wars D6
- Equipment and Armor
- Weapons
- General Goods
- Typical Gamemaster Characters
- Starfighters and their Pilots
- Assorted Helpful Macros

**Note:** Importing a character template can take up to a minute as it searches for all the skills and items.
